# Credits

The CREDITS (or AUTHORS) file lists the primary authors of the project, the maintainers, and contributors.
A funding acknowledgment may also be included.

Examples of CREDITS documents:

* the [CREDITS.md](https://github.com/csdms/bmi-topography/blob/main/CREDITS.md) file for the Topography Data Component
* Landlab has an [AUTHORS.md](https://github.com/landlab/landlab/blob/master/AUTHORS.md) file
* Conda also has an [AUTHORS.md](https://github.com/conda/conda/blob/main/AUTHORS.md) file
