# Code of Conduct

This file explains how anyone participating in this project is expected to behave.

The [Contributor Covenant](https://www.contributor-covenant.org/) is a widely used code of conduct for open source projects.
It provides guidelines for how to create a welcoming and inclusive community, and how to handle conflicts and misconduct.

Examples of CODE-OF-CONDUCT documents:

* the CSDMS project [CODE-OF-CONDUCT](https://github.com/csdms/project/blob/main/CODE-OF-CONDUCT.md) document
* the [SciPy Code of Conduct](https://docs.scipy.org/doc/scipy/dev/conduct/code_of_conduct.html)
* the [CODE-OF-CONDUCT](https://github.com/csdms/bmi-topography/blob/main/CODE-OF-CONDUCT.md) document for the Topography Data Component
